# UbuntuAppPacks
## i3-gaps
i3-gaps是fork自i3wm的改进版。</br>
安装组件
- polybar
</br>状态栏，取代i3-status，注意复制后需更改polybar.sh权限。
- NetworkManager
</br>用于管理网络，用法：
</br>nmcli：命令行。这是最常用的工具；
</br>nmtui：在shell终端开启文本图形界面；
</br>Freedesktop applet：如GNOME上自带的网络管理工具；
</br>nm-connection-editor图形配置工具。
