# UbuntuAppPacks
## i3-gaps
i3-gaps是fork自i3wm的改进版。<br>
**安装组件**
### polybar
状态栏，取代i3-status，注意复制后需更改polybar.sh权限。
### xrandr
<br>调整屏幕分辨率（系统自带）。用法：
- 查看信息：
xrandr
- 调指定分辨率：
xrandr -s 1920x1080
xrandr -s 0
xrandr -s 1600x900
xrandr -s 4
### NetworkManager
用于管理网络，用法：
- nmcli：命令行。这是最常用的工具；
- nmtui：在shell终端开启文本图形界面；
- nm-connection-editor图形配置工具。
